    \begin{itemize}
      \item Landauniveaus auf gegenüberliegenden Dirac-Punkten (unterschiedliche Subgitter!) haben das gleiche Spektrum
      \item Landauniveaus zweifach entartet
      \item Null Energie Landauniveau teilen sich Elektronen und Löcher\\
      $\rightarrow$ damit teilt es Elektronzustände ($\mu > 0 $) und Lochzustände ($\mu < 0 $)
    \end{itemize}
